# ES6 - JavaScript Study Notes

Topic-wise organized ES6+ JavaScript programs and examples from classroom learning.

## Folder Structure

```
ES6/
├── 01-Destructuring/
│   ├── 01-object-destructuring.js   — Basic & nested object destructuring
│   ├── 02-array-destructuring.js    — Array unpacking with rest operator
│   └── 03-default-values.js         — Default values for undefined properties
│
├── 02-Spread-and-Rest/
│   ├── 01-rest-parameters.js        — Accepting variable arguments in functions
│   └── 02-spread-operator.js        — Merging arrays, copying objects
│
├── 03-Array-Methods/
│   └── 01-filter-and-reduce.js      — filter() and reduce() with examples
│
├── 04-Asynchronous-JS/
│   ├── 01-setTimeout.js             — Sync vs Async, Event Loop
│   └── index.html                   — setTimeout DOM demo in browser
│
├── Practice/
│   └── student-scholarship.js       — Student eligibility checker
│
└── Python-Notes/
    ├── t2-marks-imputation.py       — Missing marks imputation
    └── t22-sales-imputation.py      — Missing sales data imputation
```

## How to Run

```bash
# Run any JS file with Node.js
node 01-Destructuring/01-object-destructuring.js

# Open HTML files directly in browser
```

## Topics Covered

1. **Destructuring** — Object, Array, Nested, Default Values
2. **Spread & Rest Operators** — Rest params, Spread for arrays/objects
3. **Array Methods** — filter(), reduce()
4. **Asynchronous JavaScript** — setTimeout, Event Loop
